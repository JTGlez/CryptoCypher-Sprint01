# Introducción

Proyecto del Sprint 01 de Oracle ONE: Cifrador y descifrador de texto plano desarrollado con HTML, CSS y JS. Esta aplicación permite cifrar y descifrar bajo 3 formas
distintas:

| Algoritmo | Tamaño |
| ------ | ------ |
| Reemplazo de letras |  --- |
| AES-GCM | 256 bits |
| ChaCha20 | 256 bits |
